# Crossword-Auto-Fill-Assistant-Javaa

## Project Description
Java program that fills a partial crossword grid by placing given words horizontally or vertically using backtracking.

## Steps to Run
1. Copy the code into `Main.java`
2. Open terminal / command prompt
3. Compile: `javac Main.java`
4. Run: `java Main`

## Input Format
- Grid: 2D char array (`.` = empty, `#` = block, letters = pre-filled)
- Words: String array of words to place

## Output Explanation
Program prints every valid solution where all words are placed correctly without conflicts.
